# Mapa de navegacion principal - FIS EPN

Documento vivo para documentar la navegacion publica principal del sitio FIS de la EPN.

## 1) Alcance
Este documento resume la estructura de navegacion visible en la pagina inicial y en las paginas principales del sitio.

Criterio usado:
- Enfoque en enlaces principales de menu, cabecera, breadcrumbs y bloques de navegacion visibles.
- Se prioriza lo que el usuario puede seguir desde la home.
- Las paginas complejas se listan como paginas madre con sus hijos directos visibles.

## 4) Observaciones de navegacion importantes
- La home funciona como mapa de entrada a todas las secciones principales.
- La navegacion esta distribuida en:
  - menu superior,
  - menu movil/colapsado,
  - breadcrumbs,
  - bloques laterales o de contenido.
- `Planta docente` es una seccion especialmente importante porque concentra muchas fichas individuales y paginacion.
- `Gestion de Transferencia Tecnologica` tambien actua como nodo con varias subpaginas.
- `Posgrado` y `Investigacion` concentran contenido con mucho texto estructurado y varias paginas hijas.
